# MarkovSeqRec
Simple and fast tool for Markov chain sequential recommendation

## [`build.py`](build.py) — Build Markov Chain
First, use [`build.py`](build.py) to build a Markov chain from an input CSV/TSV file containing user inspection data.

## [`recommend.py`](recommend.py) — Recommend Items
Next, use [`recommend.py`](recommend.py) to recommend items for users.
